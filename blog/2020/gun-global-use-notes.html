安装

由于我使用的是debian系统 

sudo apt-get install global

使用

生成文件

$ cd source_dir
$ gtags -v #目录下会多了三个文件 GTAGS(定义数据库),GRTAGS(引用数据库),GPATH(路径名数据库)
